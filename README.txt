
CoreIndia News Website + CMS Setup Instructions
------------------------------------------------

1. Upload all files to a new GitHub repository.
2. Enable GitHub Pages on the main branch.
3. Access site via https://yourusername.github.io/repo-name/

Admin Panel Login:
- Username: admin
- Password: 1234
- Open admin.html locally or host it to use.

Google Sheet Backend (to be connected manually):
- Create Google Sheet with columns: Title | Category | ImageURL | Content | Date
- Use Google Apps Script and Web App Deployment.
- Add fetch/POST logic to admin.html (example provided in documentation).
